Game objective survive as long as you can

to join just press any button on keyboard/gamepad

to move use the left stick or WASD
to aim use the right stick or IJKL / arrow keys
to pick up weapons use right trigger or space if you have a weapon you will attack with it 
to drop weapons use button east or ctrl
start/escape to pause

enemies get stronger over time special strong enemies will start spawning after a while

known bugs
when you click play timer will run before you join the game 
pause menu buttons don't work
